# LineDogParkour 游戏说明

## 游戏简介
LineDogParkour 是一款基于 Qt 的 2D 跑酷游戏，玩家需要控制一只小狗在充满障碍物的场景中奔跑、跳跃并收集金币。游戏的目标是尽可能长时间地存活并获得高分。

## 游戏玩法
1. **移动与跳跃**：
   - **按下空格键**：小狗立即跳跃，获得最大跳跃高度。
   - **快速松开空格键**：可以控制跳跃高度，松开得越早，跳跃越低（最低为系统设定的最小高度）。
   - **长按空格键**：保持最大跳跃高度。
2. **收集金币**：
   - 在奔跑过程中尽量收集金币以增加分数。
3. **躲避障碍**：
   - 避开场景中的障碍物，否则失去一条生命。
   - 失去全部生命后游戏结束。

## 游戏目标
- 获得尽可能高的分数。
- 挑战自己的反应速度和操作技巧。

## 游戏特色
- **精美的像素风格**：
  - 游戏采用了简洁的像素风格，画面清新。
- **动态场景**：
  - 随机生成的障碍物和金币位置，增加了游戏的可玩性。
- **灵敏的跳跃控制**：
  - 通过按键时长控制跳跃高度，实现精准跳跃。
  - 玩家可以快速反应，躲避障碍或收集金币。
- **简单易上手**：
  - 操作简单，适合所有年龄段的玩家。
- **生命系统**：
  - 初始 3 条生命，碰撞障碍物会失去一条生命。

## 开发者指南
### 项目结构
- **资源文件**：
  - 所有游戏素材存放在 `images/` 目录下，并通过 `resources.qrc` 注册。
- **主要代码文件**：
  - `DogItem.cpp`：小狗的逻辑实现（移动、跳跃、动画）。
  - `Obstacle.cpp`：障碍物的逻辑实现。
  - `Coin.cpp`：金币的逻辑实现。
  - `MainWindow.cpp`：游戏主窗口、场景管理和碰撞检测。
  - `Config.h`：游戏配置常数（物理参数、UI 参数等）。
- **核心机制**：
  - **跳跃系统**：支持可控高度的跳跃。长按保持高度，快速松开降低高度。
    - `kJumpImpulse`：跳跃初始速度。
    - `kJumpReleaseMultiplier`：松开时的速度衰减比例。
    - `kMinJumpVelocity`：最小跳跃速度（保证最低高度）。
  - **生命与碰撞**：玩家从 3 条生命开始，每碰撞一次障碍物失去一条。
  - **分数系统**：收集金币增加分数。

### 构建与运行
1. 确保安装了 Qt 和 CMake。
2. 在项目根目录下运行以下命令：
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ./LineDogParkour
   ```
