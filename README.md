# LineDogParkour - 小狗跑酷游戏

## 游戏简介

LineDogParkour 是一款基于 Qt 的 2D 横版跑酷游戏，采用马里奥式跳跃系统，玩家需要控制一只可爱的小狗在充满障碍物和惊喜的关卡中奔跑、跳跃、收集金币并顶开问号方块。

当达到特定分数（1000/2000/3000）时，会触发经典的**推箱子（Sokoban）解谜关卡**！完成推箱子关卡后可以继续跑酷冒险。

游戏的目标是尽可能长时间地存活、获得高分，并挑战所有推箱子关卡。

## 核心玩法

### 跑酷模式（Runner Mode）

#### 基础操作

| 按键 | 功能 | 说明 |
|------|------|------|
| **空格** | 跳跃 | 短按=小跳，长按=大跳 |
| **Shift** | 奔跑 | 按住加速，跳得更远 |
| **A / ←** | 左移 | 向左移动 |
| **D / →** | 右移 | 向右移动 |
| **R** | 重新开始 | 游戏结束后重新开始 |

### 推箱子模式（Sokoban Mode）

#### 触发条件
- **1000分**：触发关卡1（双球入门）
- **2000分**：触发关卡2（垂直长廊挑战）
- **3000分**：触发关卡3（复杂迷宫）

#### 触发流程
1. 达到指定分数时，前方会出现一个**毛线筐**
2. 碰撞毛线筐后，小狗跳入筐中（缩小动画）
3. 淡入切换到推箱子模式

#### 推箱子操作

| 按键 | 功能 |
|------|------|
| **↑ / W** | 向上移动 |
| **↓ / S** | 向下移动 |
| **← / A** | 向左移动 |
| **→ / D** | 向右移动 |
| **ESC** | 退出推箱子模式 |

#### 推箱子规则
- **目标**：将所有毛线球（箱子）推到目标点（黄色圆圈）
- **移动**：基于网格的步进式移动，无重力
- **推动**：只能推，不能拉
- **限制**：不能推两个箱子，不能推到墙壁
- **完成**：所有箱子都在目标点上时，自动返回跑酷模式

#### 视觉反馈
- **毛线球颜色**：
  - 粉色 = 未完成
  - 绿色 = 已在目标点上
- **目标点**：半透明黄色圆圈
- **进度显示**：实时显示"已完成/总数"
- **平滑动画**：移动时有插值动画

### 马里奥式跳跃系统（跑酷模式）

#### 1. 变长跳跃
- **短按空格**：小跳（约100像素高），用于躲避低矮障碍
- **长按空格**：大跳（约200-250像素高），用于跨越深渊或上高台
- **跳跃控制**：松开空格键时，如果还在上升，速度会立即衰减

#### 2. 非对称重力
- **上升时**：重力较轻（1800），产生滞空感，方便空中调整方向
- **下落时**：重力较重（2800），快速落地，避免"太空漫步"

#### 3. 助跑加速
- **普通移动**：速度 300，跳跃距离中等
- **按住 Shift**：速度 500，跳跃距离极远
- **空中控制**：起跳后仍可调整方向，但控制力比地面弱

#### 4. 踩踏机制
- **从上方踩踏障碍物**：消灭敌人，获得 50 分
- **普通踩踏**：自动向上弹起（-400 速度）
- **按住空格踩踏**：借力大弹跳（-700 速度），可连续跳跃

#### 5. 土狼时间（Coyote Time）
- 走出平台边缘后 **120 毫秒**内仍可起跳
- 极大提升容错率，不会因为"按晚一点点"而掉坑

### 游戏目标

1. **存活**：避开障碍物，不要掉入坑洞
2. **收集金币**：增加分数
3. **顶问号方块**：获得大量金币和分数
4. **踩踏敌人**：消灭障碍物并获得额外分数
5. **挑战高分**：尽可能跑得更远

## 游戏元素

### 跑酷模式元素

#### 角色

#### 小狗（玩家）
- **尺寸**：96x96 像素
- **初始生命**：3 条
- **能力**：跑、跳、踩踏、顶方块

### 平台与方块

#### 1. 地面（Ground）
- **类型**：基础平台
- **特点**：可以站立和奔跑
- **位置**：Y = 500

#### 2. 砖块（Block）
- **类型**：普通方块
- **尺寸**：64x64 或 128x64
- **特点**：可以站立，可以从下方顶起
- **用途**：搭建多层平台

#### 3. 问号方块（Question Block）
- **尺寸**：96x96 像素
- **特点**：金黄色，中间有问号
- **功能**：从下方顶开后弹出金币
- **奖励**：+100 分
- **状态变化**：顶开后变灰色，不再有物品

### 收集物

#### 金币（Coin）
- **尺寸**：48x48 像素
- **类型1**：场景金币（静态）
  - 奖励：+10 分
  - 可以收集
- **类型2**：弹出金币（动画）
  - 来源：顶开问号方块
  - 奖励：+100 分（顶方块时获得）
  - 特点：向上弹出，划出抛物线，800ms 后消失
  - 纯视觉效果，不可收集

### 障碍物

#### 障碍物（Obstacle）
- **尺寸**：64x48 像素
- **位置**：地面上（Y = 452）
- **碰撞效果**：
  - 侧面碰撞：-1 生命
  - 从上方踩踏：消灭障碍物，+50 分

### 推箱子模式元素 

#### 毛线筐（Basket）
- **尺寸**：80x80 像素
- **特点**：棕色筐子，轻微浮动动画
- **功能**：触发推箱子关卡的道具
- **位置**：达到指定分数后在前方生成
- **提示**：显示"解谜关卡"文字

#### 墙壁（Wall）
- **尺寸**：64x64 像素
- **特点**：灰色砖墙
- **功能**：阻挡移动，不可通过
- **符号**：# （在关卡数据中）

#### 目标点（Target）
- **尺寸**：64x64 像素
- **特点**：半透明黄色圆圈
- **功能**：毛线球需要推到这里
- **符号**：. （在关卡数据中）

#### 毛线球（Box）
- **尺寸**：64x64 像素
- **颜色**：
  - 粉色：未在目标点上
  - 绿色：已在目标点上
- **功能**：需要推到目标点的箱子
- **符号**：B （在关卡数据中）

#### 小狗（推箱子模式）
- **尺寸**：64x64 像素
- **移动**：基于网格的步进式移动
- **能力**：推动毛线球
- **符号**：P （在关卡数据中）

## 关卡系统

### 区块生成

游戏采用**程序化生成**，由 5 种不同类型的区块随机组合：

#### 1. 平原区（Plain Chunk）
```
特点：简单地面，适合新手
元素：
- 地面平台
- 50% 概率：问号方块（Y=308，可从地面起跳顶到）
- 2-5 个：随机金币
- 33% 概率：障碍物
```

#### 2. 坑洞区（Pit Chunk）
```
特点：需要跳跃跨越坑洞
元素：
- 坑洞前后的地面
- 2 个：悬浮砖块（Y=300，帮助跨越）
- 1 个：空中金币
挑战：跳跃距离和时机
```

#### 3. 平台区（Platform Chunk）
```
特点：多层平台，垂直挑战
元素：
- 地面
- 2 个：低层砖块（Y=372）
- 2 个：中层砖块（Y=268）
- 1 个：问号方块（Y=84，需在中层起跳）
- 3 个：金币
挑战：跳跃高度和精准度
```

#### 4. 阶梯区（Stair Chunk）
```
特点：爬升挑战
元素：
- 4 个：阶梯砖块（逐级上升）
- 1 个：高台
- 1 个：下降砖块
- 4 个：金币
挑战：连续跳跃和节奏
```

#### 5. 问号方块区（Question Chunk）
```
特点：奖励区域
元素：
- 4 个：问号方块（Y=268，一排）
- 2 个：普通砖块
- 33% 概率：障碍物在前方（X: 50-150）
- 33% 概率：障碍物在后方（X: 750-900）
- 33% 概率：无障碍物
挑战：连续顶方块
```

### 智能生成系统

#### 自动生成
- **频率**：每 2 秒尝试生成 1 个元素
- **类型**：50% 金币，50% 障碍物
- **智能避让**：
  - 问号方块周围 150px 不生成障碍物
  - 砖块周围 100px 不生成重叠元素
  - 靠近问号方块时，障碍物变金币

#### 区块清理
- 相机左侧 1000px 外的元素自动清理
- 避免内存泄漏
- 保持性能

## 游戏机制

### 碰撞检测

#### 平台碰撞
- **从上方落下**：站立在平台上
- **从下方顶到**：触发方块效果（问号方块弹出金币）
- **从侧面碰撞**：阻挡移动

#### 物品碰撞
- **金币**：使用缩减碰撞盒（10%），更宽容的判定
- **障碍物**：使用完整碰撞盒，准确的判定

### 相机系统
- **跟随模式**：相机跟随玩家向右移动
- **触发点**：屏幕宽度的 40% 位置
- **单向移动**：只向右，不向左（经典跑酷设计）

### 分数系统

#### 跑酷模式

| 行为 | 分数 |
|------|------|
| 收集场景金币 | +10 |
| 顶开问号方块 | +100 |
| 踩踏障碍物 | +50 |
| 向右移动 | 距离 / 10 |

#### 推箱子模式
- 推箱子关卡不计分
- 完成关卡后返回跑酷模式继续计分

### 生命系统
- **初始生命**：3 条
- **失去生命**：侧面碰撞障碍物
- **游戏结束**：生命归零或掉入坑洞

## 游戏技巧

### 跑酷模式技巧

#### 新手技巧

1. **掌握跳跃**
   - 练习短跳和长跳的区别
   - 学会在空中调整方向

2. **利用助跑**
   - 按住 Shift 可以跳得更远
   - 冲刺起跳能跨越大坑洞

3. **踩踏敌人**
   - 从上方跳下踩踏障碍物
   - 按住空格踩踏可以弹得更高

4. **顶问号方块**
   - 从下方跳起顶方块
   - 每个方块只能顶一次

### 进阶技巧

1. **土狼时间利用**
   - 走出平台边缘后 0.12 秒内仍可起跳
   - 可以在边缘起跳跨越更远距离

2. **连续踩踏**
   - 踩踏敌人时按住空格
   - 可以连续高跳，快速前进

3. **精准小跳**
   - 快速点击空格
   - 用于通过狭窄空间或踩踏低矮敌人

4. **冲刺大跳**
   - Shift + 空格 + 长按
   - 可以跨越超大距离

5. **空中微调**
   - 起跳后仍可调整方向
   - 用于精准落点

### 推箱子模式技巧 

#### 基础技巧

1. **规划路线**
   - 推箱子前先想好路线
   - 避免把箱子推到死角

2. **从外到内**
   - 先推外围的箱子
   - 最后处理中心区域

3. **利用墙壁**
   - 墙壁可以作为支点
   - 帮助调整箱子位置

4. **试错学习**
   - 按 ESC 可以退出重来
   - 多次尝试找到最优解

#### 进阶技巧

1. **关卡1（双球入门）**
   - 先推右边的箱子到右上角
   - 再推左边的箱子到左边目标点
   - 注意不要把箱子推到角落

2. **关卡2（垂直长廊）**
   - 从下往上推，避免堵住通道
   - 注意中间的墙壁障碍
   - 合理利用空地进行调整

3. **关卡3（复杂迷宫）**
   - 利用对称性，分区域完成
   - 注意中央的墙壁会限制移动路径
   - 先完成一侧，再处理另一侧
   - 避免把箱子推到死角

## 难度设计

### 难度曲线
- **前期**：平原区多，障碍物少，适合熟悉操作
- **中期**：平台区和坑洞区增多，需要跳跃技巧
- **后期**：区块组合复杂，需要精准操作

### 平衡机制
- 砖块密集区域自动减少障碍物生成
- 问号方块区域无障碍物干扰
- 土狼时间提供容错
